# YouTubeSearch_App 

## Introduction
A simple Android application that allows users to search for videos using the **YouTube Data API** and display results with **RecyclerView**.  
Modern libraries such as Retrofit, Gson, and Glide are used to simplify network requests, data handling, and image loading.

=========================================================================

## Features
- Enter a search query and instantly view results.
- Use **RecyclerView** for scrolling and displaying a list of videos.
- Load video thumbnails using **Glide**.
- Handle errors gracefully (no results / network error).

=========================================================================

## Tools and Libraries
- **AndroidX AppCompat** (basic UI components).
- **Material Design Components** (modern UI elements).
- **Retrofit + OkHttp** (API communication).
- **Gson** (convert JSON into Java objects).
- **Glide** (image loading).
- **RecyclerView** (list display).
- **JUnit / Espresso** (testing).
